Class WillPaginate::LinkRenderer
In: vendor/plugins/will_paginate/lib/will_paginate/view_helpers.rb
Parent: Object

This class does the heavy lifting of actually building the pagination links. It is used by will_paginate helper internally, but avoid using it directly (for now) because its API is not set in stone yet.
